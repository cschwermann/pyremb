public subroutine Mol_interpolate(molecule, ingrid)

Interpolation of all functions in a molecule onto some grid.

This subroutine interpolates density, gradient and nuclear potential of a molecule onto a given grid. An implementation of Shepards method is used.

type(molecule_t), intent(inout) :: molecule

Input: molecule, will be modified
