A Subsystem DFT package able to combine arbitrary - periodic and non-periodic - molecular systems

PEREMB

Usage

PEREMB uses densities (as well as other molecular information, e.g. density gradients, nuclear potential or nuclear positions) of multiple subsystems as input and returns an effective embedding potential generated from inactive subsystems acting on an active subsystem as output.
